## Critical Rules

1. **100 replies/day minimum** — This is how you grow from 0
2. **Never post without a visual** — Screenshots, demos, code
3. **One sentence per line** — Makes tweets scannable
4. **Use "I" not "you"** — Share your journey, don't lecture
5. **Engage before promoting** — Build relationships first
6. **Show results, not promises** — Ship updates, share metrics
